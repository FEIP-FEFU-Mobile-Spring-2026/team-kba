# team-kba — Mobile Store


## Стек технологий
- **Платформа:** Android
- **Язык:** Kotlin
- **UI:** XML layouts, Material Design 3
- **Загрузка изображений:** Glide
- **Асинхронность:** Coroutines + Flow
- **Архитектура:** MVVM (ViewModel + Repository)

## Реализованный функционал (Блок 1)
-  Загрузка списка товаров из JSON-файла
-  Отображение карточек товаров (изображение, название, описание, цена)
-  Фильтрация по категориям (Новинки, Джинсы, Футболки, и т.д.)
-  Адаптация под поворот экрана
-  Обработка состояний загрузки и ошибки
