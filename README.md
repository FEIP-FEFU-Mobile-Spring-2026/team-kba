# team-kba — Mobile Store

## О проекте
Приложение интернет-магазина одежды. Реализован экран каталога с загрузкой товаров из JSON, фильтрацией по категориям и обработкой состояний загрузки/ошибки.

## Стек технологий
- **Платформа:** Android
- **Язык:** Kotlin
- **UI:** XML layouts, Material Design 3
- **Загрузка изображений:** Glide
- **Асинхронность:** Coroutines + Flow
- **Архитектура:** MVVM (ViewModel + Repository)

## Реализованный функционал (Блок 2)

### Данные
- [x] Загрузка товаров из `products.json` через Repository
- [x] Категория «Новинки» формируется на клиенте (товары с тегом `"New"`)
- [x] Категория «Новинки» отображается первой в списке табов

### Каталог
- [x] Переключение табов фильтрует товары по категории
- [x] Карточка товара соответствует макету (отступы, шрифты, цвета, скругления)
- [x] Карточка отображает: изображение, название, цену (формат: `1 234 ₽`)
- [x] Индикатор загрузки (ProgressBar) при старте
- [x] Состояние ошибки с кнопкой повтора

### Навигация
- [x] Нижнее меню с двумя пунктами: каталог и корзина (заглушка)

### Архитектура
- [x] Данные загружаются через Repository (не напрямую в ViewModel)

### Lifecycle
- [x] Поворот экрана не крашит приложение и не сбрасывает состояние
- [x] Убийство процесса → восстановление состояния (SavedStateHandle)

## Как запустить
1. Клонировать репозиторий
2. Открыть проект в Android Studio
3. Синхронизировать Gradle
4. Запустить на эмуляторе или устройстве (minSdk = 24)
